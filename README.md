# Simple React Calculator
Simple React Calculator created for learning basics of React.

## Features
- Simple operation (+-/*) beetween two operands
- Allow to use keyboard (numpad, to calculate use *Enter*, to clear use *Backspace*)

## Setup
Dependencies:
```
$ npm install
// or
$ yarn
```
Dev mode:
```
$ npm run start
// or
$ yarn start
```
Tests:
```
$ npm run test
// or
$ yarn test
```
